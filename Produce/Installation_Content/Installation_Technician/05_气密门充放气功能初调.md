# Airtight Door Inflation-Deflation Functional Initial Check: Verifying Seal Operation Before System Handover

## 适用受众
Installation_Technician

## 写作角度说明
气密门充放气功能初调是验证密封系统是否正常工作的关键步骤。安装技师需要在系统上电后、正式调试前，先对每扇门进行独立的手动功能测试，确认充气压力、密封状态和视觉指示灯正常。

## 核心论点 / 洞察
Testing the airtight door with the frame seal only — without the pneumatic seal inflated — misses the primary failure mode: the door appears sealed but the inflatable gasket is not engaging.

## 参考素材
- Inflation pressure verification: read gauge at pneumatic seal inlet, confirm ≥0.25 MPa (manufacturer spec), compare against PLC display value
- Visual indicator check: red LED when door unlocked/unsealed, green LED when seal inflated and interlock satisfied
- Cycle time verification: inflation time ≤5 seconds, deflation time ≤5 seconds per product spec, measure with stopwatch
- Interlock behavior test: door cannot open when seal is not inflated (test by manually blocking seal), alarm sounds when pressure drops below 0.15 MPa
- EMI/interlock sequence: with seal inflated and door closed, trigger interlock input, confirm door remains locked, record sequence timing
