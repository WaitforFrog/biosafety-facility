# On-Site Pressure Decay Test Verification: ASTM E779 Method Application for Airlock Containment Validation

## 适用受众
Commissioning_Engineer

## 写作角度说明
压差梯度现场验证是确认气密门和传递窗密封性能的核心测试。调试工程师需要按照ASTM E779标准方法执行压力衰减测试，记录压降数据，判定密封性能是否满足设计要求，并出具正式的验证报告。

## 核心论点 / 洞察
Performing a pressure decay test with the door unseated — testing only the frame seal — misses the full sealing system failure mode that occurs during actual inflation-deflation operation.

## 参考素材
- ASTM E779-10 test method: fill enclosure to 250 Pa above ambient, isolate, measure decay over 1-minute interval, calculate air leakage rate in L/s at 25 Pa
- Test configuration: door in operational (inflated) condition, all openings sealed, one differential pressure gauge inside, one reference gauge outside, minimum 3 test runs per door
- Acceptance criteria: biosafety level 3 enclosures: ≤0.05 L/s at 25 Pa per ASTM E779; biosafety level 2: ≤0.1 L/s at 25 Pa
- Equipment requirements: calibrated differential pressure gauge (resolution 0.1 Pa), flow measurement hood (for exhaust flow verification), temperature logging (±1°C accuracy)
- Documentation requirements: as-found and as-left data, environmental conditions (temperature, barometric pressure), test equipment calibration certificates
