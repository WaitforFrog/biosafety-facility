# HEPA Filter Field Installation and In-Situ Leak Testing: DOP/PAO Scan Procedure for Installed Filters

## 适用受众
Installation_Technician

## 写作角度说明
高效过滤器（HEPA）的现场安装与检漏是洁净区安装的最后关键步骤之一。安装技师需要在过滤器安装后立即进行现场检漏测试（DOP/PAO扫描法），确保过滤器本体的密封安装质量，防止旁通泄漏。

## 核心论点 / 洞察
Scanning only the filter face — without extending the probe along the filter frame gasket seam — misses bypass leakage through improperly seated filter frames, the most common HEPA installation failure.

## 参考素材
- DOP/PAO aerosol generator: TSI AeroTrak or equivalent, aerosol challenge concentration 10-100 μg/L upstream, aerosol type PAO-4 or DEHS
- Downstream detection: metered sampling probe with laser particle counter, minimum sample flow 28.3 L/min (1 CFM), probe traverse speed 25-50 mm/second
- Scan pattern: 25 mm grid traverse pattern across entire filter face and frame perimeter, including seal joints
- Acceptance criterion: no single point reading >0.01% of upstream challenge concentration (per IEST-RP-CC001), scan result ≤0.01% penetration
- Filter handling: install with arrow on filter frame pointing in direction of airflow, handle by frame only (never by media), verify gasket integrity before insertion
