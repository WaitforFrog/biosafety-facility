# Interlock Controller Installation and Initial Configuration: Field Programming of Safety Interlock Logic

## 适用受众
Installation_Technician

## 写作角度说明
互锁控制器的安装与初始配置是气密门和传递窗控制系统调试的第一步。安装技师需要完成控制器的机械安装、供电接线，并配合调试工程师完成基本的逻辑配置和现场适配。

## 核心论点 / 洞察
Programming interlock logic in the factory — without on-site verification of sensor operation and door travel distances — produces logic parameters that require full reconfiguration upon commissioning.

## 参考素材
- Controller mounting: DIN rail or panel mount in accessible location, minimum IP54 enclosure rating, ambient temperature rating 0-45°C
- Power supply: 24V DC nominal (18-32V operating range), reverse polarity protection required, power consumption typically 5-15 W per controller
- Sensor inputs: door position sensors (proximity switches or magnetic reed), seal pressure switches (NAMUR or voltage-free contacts), emergency stop inputs
- Output configuration: solenoid valve drivers (24V DC, current rating 1-3 A), indicator lamp outputs, BMS alarm outputs
- Initial configuration parameters: door close confirmation time delay (typically 0.5-2 seconds), seal inflation timeout (typically 5-10 seconds), alarm delay timers
- Programming interface: USB or Ethernet configuration port, handheld HMI panel, or laptop with manufacturer software
