# Installation Self-Inspection and Issue Recording: Punch List Management and Pre-Commissioning Verification

## 适用受众
Installation_Technician

## 写作角度说明
安装自检与问题记录是安装技师在完成安装工作后的自查和文档化工作。通过系统性的自检清单和现场问题记录，确保安装质量可控，问题可追溯，为后续调试和验收提供完整的安装状态文档。

## 核心论点 / 洞察
Treating the punch list as a commissioning document — rather than an installation quality record — means that resolved installation defects are never formally closed, creating liability ambiguity during the warranty period.

## 参考素材
- Pre-commissioning checklist items: all mechanical fixings torqued and marked, all electrical connections verified tight, all seals inspected and undamaged, all equipment cleaned and protected, all documentation handed over
- Punch list format: structured database entry with item number, location, description, severity (critical/major/minor), responsible party, target date, resolution date, resolution evidence photo
- Severity classification: critical = prevents commissioning (e.g., unanchored equipment), major = affects performance (e.g., misaligned door), minor = cosmetic/functional (e.g., scratched surface)
- Sign-off requirements: installation technician self-sign-off, site supervisor counter-sign-off, commissioning engineer pre-start acceptance
- Record retention: punch list and resolution records retained for minimum 10 years, linked to equipment serial number and installation date
