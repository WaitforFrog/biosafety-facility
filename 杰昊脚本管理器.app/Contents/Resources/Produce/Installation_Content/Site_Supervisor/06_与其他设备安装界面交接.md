# Other Equipment Installation Interface Coordination: Managing Mechanical and Electrical Interface Boundaries Between Biosafety Equipment and Adjacent Systems

## 适用受众
Site_Supervisor

## 写作角度说明
与其他设备安装界面交接是现场主管管理多系统交叉施工的核心工作。生物安全设备与HVAC、给排水、消防等系统存在多个物理和电气接口，接口区域的安装顺序和配合要求需要专门协调。

## 核心论点 / 洞察
The interface between a biosafety pass box and the HVAC duct is the single most contested installation boundary — neither the HVAC contractor nor the equipment installer claims responsibility for the duct-to-flange sealing detail, resulting in systematic leakage at this joint.

## 参考素材
- Interface identification: list all physical interfaces (duct connections, conduit entries, drain connections, structural penetrations) with defined responsible party per interface agreement
- Interface responsibility matrix: define who supplies sealing materials, who applies sealant, who provides temporary protection during other trades' work, who inspects after work is complete
- Sequential interface agreement: document the sequence in which each trade works at each interface point, establish which trade finishes first and which follows
- Interface inspection: each interface joint requires joint inspection before being covered up or concealed, photograph each completed interface joint
- Warranty implication: any interface joint not inspected and documented becomes the responsibility of the last covering trade, not the responsible installing trade
