# Power Load and Grounding Requirements: Calculating Electrical Demand and Establishing Equipotential Bonding for Biosafety Equipment

## 适用受众
Subcontractor

## 写作角度说明
电源负载与接地要求是电气分包商计算设备电力需求和配置保护接地的核心工作。生物安全设备的电力负载计算和接地系统设计关系到设备安全运行和人员防护，必须严格按照设计规范执行。

## 核心论点 / 洞察
Sizing the supply cable based only on the equipment nameplate full-load current — without accounting for inrush current (which can be 3-5× running current for motors and solenoid coils) — risks voltage drop during startup that causes nuisance control system resets.

## 参考素材
- Load calculation: running power (W) / voltage (V) = full-load current (A) → apply demand factor (0.8 for multiple similar loads) → apply diversity factor → select cable cross-section per IEC 60364
- Inrush current: solenoid valve inrush typically 3-5× holding current, duration 50-100 ms → motor inrush 5-7× FLA, duration 1-3 seconds → use soft-start or star-delta for motors >5 kW
- Protective device sizing: circuit breaker or fuse rating = 1.25 × full-load current (IEC standard) → selectivity coordination with upstream protective device
- Grounding system: protective earth (PE) conductor per IEC 60364 → equipotential bonding conductors → signal reference ground (isolated from PE for BMS communication circuits) → grounding resistance ≤0.1 Ω
- Surge protection: install Type 2 SPD (Surge Protective Device) at main distribution board → consider Type 1 SPD for buildings with lightning protection system
