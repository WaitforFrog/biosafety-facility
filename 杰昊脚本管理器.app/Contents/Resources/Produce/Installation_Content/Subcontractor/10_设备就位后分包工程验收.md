# Subcontractor Works Acceptance After Equipment Positioning: Inspecting and Signing Off Electrical and HVAC Work Upon Equipment Installation Completion

## 适用受众
Subcontractor

## 写作角度说明
设备就位后分包工程验收是电气/HVAC分包商在设备安装完成后，对各自专业范围内的安装工作进行自检并提交验收的关键节点。通过规范的验收流程，确保分包商的工作质量满足设备和业主的要求。

## 核心论点 / 洞察
The electrical subcontractor refusing to sign the acceptance form — because the BMS integration was done by a different subcontractor — creates a gap where the electrical installation is never formally accepted, leaving the electrical contractor liable indefinitely.

## 参考素材
- Pre-acceptance self-inspection checklist: all cable terminations verified tight → all cable identification labels installed → all cable trays installed with covers → conduit terminations sealed → earth resistance measured and recorded → insulation resistance tested (minimum 1 MΩ for power circuits, 0.5 MΩ for control circuits)
- Inspection and test plan (ITP): agreed ITP with client before work starts → hold points (witness points) at critical stages → sign-off at each hold point → final acceptance sign-off
- Punch list resolution: if installation work does not pass pre-acceptance inspection → issue punch list to subcontractor → set resolution deadline → re-inspect after resolution → only issue acceptance when all critical and major items resolved
- Snag list scope: label missing → terminations not torqued → cable tray cover missing → conduit seal missing → protective conduit entry bushings missing
- Handover documentation: as-built drawings (marked-up field modifications) → cable schedule (updated with actual route and length) → test results record (earth resistance, insulation resistance) → material certificates
