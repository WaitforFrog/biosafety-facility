# Project Completion Documentation and As-Built Record Handover: Compiling and Submitting All Electrical and HVAC Documentation for Final Project Closeout

## 适用受众
Subcontractor

## 写作角度说明
工程竣工资料交接清单是电气/HVAC分包商在项目竣工时提交完整竣工资料的核心工作。竣工资料是业主运维团队进行设备运维和未来改造的必要依据，资料的完整性和准确性直接影响业主的设施管理能力。

## 核心论点 / 洞察
Handing over as-built drawings without comparing them against the actual installation — relying solely on field marks on the design drawings — guarantees that some discrepancies between drawings and reality will be present, creating maintenance risk.

## 参考素材
- As-built drawing requirements: mark all deviations from design drawings in red → annotate actual cable routes, lengths, and termination points → include coordinate references for underground cables and conduits
- Cable schedule requirements: circuit reference → cable type and size → from equipment → to equipment → route reference → length → termination point at both ends
- Test result records: earth resistance test results per circuit → insulation resistance test results per circuit → continuity test results for bonding conductors → relay/breaker coordination test results
- Certification documents: IEC installation certificate (or equivalent national standard) → test reports for all inspection and testing items → calibration certificates for any test instruments used
- Submission format: submit both printed (2 copies) and electronic (PDF + native CAD format) → organize by discipline (electrical/HVAC) → index all documents → include document transmittal form
- Submission timeline: submit within 30 days of project completion → client has 14 days to review and return comments → address comments and resubmit within 14 days

## 关键词
As-Built Drawings, Project Closeout, Cable Schedule, Test Certificates, Documentation Handover, Commissioning Records
